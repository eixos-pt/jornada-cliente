# VERA — Jornada do Cliente Eixos

Página de acompanhamento da jornada do cliente na Eixos.

O cliente acompanha em tempo real o progresso da sua proposta — desde o upload de documentos até ao onboarding.

## Funcionalidades

- Upload de documentos (Modelo 22, IES, Balancete, e-Fatura, Certidão)
- Acompanhamento em tempo real (5 passos)
- Proposta personalizada com aceitar/recusar
- Contrato via DocuSign
- Onboarding automático
- Bilingue PT/EN

## Stack

- HTML + CSS + JavaScript (single page)
- n8n (webhooks backend)
- Monday.com (CRM)
- Gamma (propostas)
- DocuSign (contratos)
